Listas

En HTML existen dos tipos de listas: las listas desordenadas, o unordered lists <ul>, y las listas ordenadas, u ordered lists <ol>. Ambas, a su vez, contienen etiquetas hijas llamadas list items <li>. El uso de una u otra depende de lo que necesitemos listar: si el orden importa, usamos una lista ordenada; si el orden no es relevante, podemos usar una lista desordenada.

Ejemplo de una lista desordenada, o unordered list

  • List item 01
  • List item 02
  • List item 03
  • List item 04

Ejemplo de una lista ordenada, u ordered list

  1. List item 01
  2. List item 02
  3. List item 03
  4. List item 04
