Formularios

Los formularios nos permiten recopilar datos del usuario y enviarlos a un servidor backend, deben tener un metodo que por lo general es post, los creamos con la etiqueta <form> a la vez dentro van las etiquetas <form> con su respectivo tipo, es decir texto, correo electrónico, contraseña, fecha, rango, color, subir archivo, checkbox, radio, los inputs de texto pueden tener el atributo placeholder que es un texto que describe el tipo de campo que debe diligenciar el usuario, el atributo name que es el identificador del campo cuando el formulario se envía. Es el nombre con el que llegan los datos al servidor, también el atributo required que obliga al usuario a diligenciar el campo para poder enviar el formulario, hasta que no lo diligencia el formulario no se enviara.
