Metodología BEM

Block Element Modifier

BEM es una convención de nomenclatura para clases CSS que facilita la creación de componentes reutilizables, escalables y mantenibles en el desarrollo frontend. Organiza el código en bloques independientes, elementos dependientes y modificadores de estilo, lo que mejora la legibilidad y evita conflictos de especificidad en CSS.
