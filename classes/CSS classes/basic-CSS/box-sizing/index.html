Box-sizing

El box-sizing es una propiedad CSS que define cómo se calcula el tamaño total de un elemento, específicamente cómo se tienen en cuenta el padding y el borde en el ancho y alto declarados.

Los dos valores

content-box (valor por defecto)

El ancho y alto que definimos aplican solo al contenido. El padding y el borde se suman por fuera, haciendo el elemento más grande de lo que declaramos en ancho y alto.

border-box

El ancho y alto que definimos incluyen el padding y el borde. El contenido se encoge para acomodarlos, pero el elemento ocupa exactamente lo que declaramos.

Ejemplo

Caja con content-box

Declaramos un ancho y alto de 100px, le damos un padding de 20px, y un borde de 2px, es decir la caja medira 122pxpx de alto y ancho, ya que el padding y border se suman

Caja con border-box

Declaramos un ancho y alto de 100px, le damos un padding de 20px, y un borde de 2px, damos la propiedad box-sizing: content-box, ahora la caja medira 100px y se adaptara por mas de que le demos padding y border para medir exactamente lo que declaramos en width y height.
