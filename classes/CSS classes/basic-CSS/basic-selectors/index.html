Selectores básicos

Los selectores en CSS, tienen tres formas principales de apuntar a elementos, por elemento, o etiqueta, que dara estilos a todos los elementos con esa etiqueta, por clase(.clase), que dara estilos a todos los elementos con esa clase, ueden haber varios elementos con la misma calse en la página o por id (#id), que aplica estilos a un único elemento (los IDs deben ser únicos en la página).
