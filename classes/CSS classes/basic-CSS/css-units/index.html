    Unidades absolutas

  • pixeles: Píxel--La más común en pantallas
  • cm, mm, in: Centímetros, milímetros, pulgadas — útiles para impresión
  • pt: Puntos (1pt = 1/72 de pulgada)
  • pc: Picas (1pc = 12pt)

    Unidades relativas

    Son las más poderosas y las preferidas en diseño moderno porque se adaptan al contexto.

  • em: Relativa al font-size del elemento padre
  • rem: Relativa al font-size del elemento raíz html
  • Porcentaje:Relativa al tamaño del elemento padre

    Relativas al viewport

  • vw: 1% del ancho del viewport
  • vh: 1% del alto del viewport
  • vmin: 1% del lado menor del viewport
  • vmax: 1% del lado mayor del viewport

    Cuando usar cada una

  • px: Bordes, sombras, detalles finos.
  • rem: Tipografía y espaciado consistente en toda la app
  • em: Padding/margin proporcional al texto del componente
  • %: Layouts fluidos
  • vw/vh: Secciones de pantalla completa, tipografía fluida
