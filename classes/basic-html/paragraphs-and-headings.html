La etiqueta <h1> representa el título principal de la página web, solo debe haber un h1 por página, y debe ser lo más descriptivo y representativoposible.

La etiqueta <h2> representa subtítulos importantes dentro de h1, de h2 a h6 podemos usar varias veces las etiquetas

<h3>: Subtemas dentro de h3
