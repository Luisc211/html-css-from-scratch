Rutas

En HTML existen dos tipos de rutas, las rutas relativas, y las rutas absoultas, las rutas relativas son aquellas que para citarlas las llamamos d euna u otra forma, según donde estemos ubicados, mientras que las rutas absolutas siempre seran las mismas para citarlas, no importa donde estemos, pueden ser imágenes, audios, videos, archivos CSS, scripts de JavaScript, las rutas absolutas se suelen ustilizar para consumir recursos externos, es decir de internet, u otros servidores, mientras que las rutas relativas son recursos que consumimos de nuestro propio proyecto, podemos consumir rescursos dentro de la misma carpeta, ir a una subcarpeta, subir de nivel, o consumir elementos de carpeta padre, o dos carpetas padres.
